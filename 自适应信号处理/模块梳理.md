### 1.3 线性滤波器算法设计

### 一、 随机梯度法 (Stochastic Gradient Method, 代表算法: LMS)

#### 1. 原理

- **优化目标**：最小化均方误差（MSE），即 J(w)=E[∣e(n)∣2]*J*(*w*)=*E*[∣*e*(*n*)∣2]，这是一个**统计意义**上的目标函数。
- **核心思想**：由于在实时处理中无法得到信号的精确统计特性（即无法计算真实的梯度 ∇J(w)∇*J*(*w*)），随机梯度法使用**当前单一采样点的瞬时误差平方**作为均方误差的估计，从而得到瞬时梯度。
- **更新公式**（LMS）：w(n+1)=w(n)+μ⋅e(n)⋅x(n)*w*(*n*+1)=*w*(*n*)+*μ*⋅*e*(*n*)⋅*x*(*n*)
  - 其中 μ*μ* 是步长因子，e(n)*e*(*n*) 是误差，x(n)*x*(*n*) 是输入向量。
- **特点**：基于最速下降法，每次迭代沿着目标函数梯度估计的负方向调整权重。

#### 2. 应用

- 由于计算复杂度极低，特别适用于**计算资源受限、要求实时处理**的场景。
- 典型应用：回声消除（AEC）、主动降噪（ANC）、通信系统中的自适应均衡器、生物医学信号去噪等。

------

### 二、 最小二乘估计法 (Least Squares Estimation, 代表算法: RLS)

#### 1. 原理

- **优化目标**：最小化加权误差平方和，即 ϵ(n)=∑i=1nλn−i∣e(i)∣2*ϵ*(*n*)=∑*i*=1*n**λ**n*−*i*∣*e*(*i*)∣2，这是一个**确定性**的目标函数（基于已观测到的全部数据）。
- **核心思想**：利用截至目前的所有输入数据，寻找一组权重，使得历史误差的加权平方和最小。为了适应非平稳过程，引入了遗忘因子 λ*λ*（通常接近1，如0.99），使得新数据的权重高于老数据。
- **更新机制**：利用矩阵求逆引理，递归地更新自相关矩阵的逆 P(n)=[∑λn−ix(i)xH(i)]−1*P*(*n*)=[∑*λ**n*−*i**x*(*i*)*x**H*(*i*)]−1，从而直接计算出最优权重，无需梯度下降。
- **特点**：等价于在每一步都求解了一个最优的维纳滤波器（基于当前数据窗）。

#### 2. 应用

- 由于收敛极快且对输入信号特征值不敏感，适用于**对收敛速度要求极高、或输入信号高度相关（特征值分散大）**的场景。
- 典型应用：雷达/声纳阵列自适应波束形成、高精度系统辨识、快速时变信道的均衡、CDMA系统中的多用户检测等。

------

### 三、 两者的异同

#### 1. 相同点（同）

1. **目标一致**：都是自适应滤波算法，目标都是动态调整滤波器权重以最小化误差信号。
2. **基本结构**：都依赖于横向滤波器（FIR）结构，都需要期望信号 d(n)*d*(*n*) 作为参考。
3. **递归特性**：都是在线算法，每接收一个新样本，就更新一次权重。

#### 2. 不同点（异）

| 对比维度             | 随机梯度法 (LMS)                                             | 最小二乘法 (RLS)                                             |
| :------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **优化准则**         | **统计平均**（MSE，均方误差）                                | **时间平均**（LS，误差平方和）                               |
| **信息利用**         | 仅利用**当前时刻**的数据（瞬时梯度）                         | 利用从开始到**当前所有时刻**的数据（带遗忘因子）             |
| **收敛速度**         | **慢**。尤其当输入信号特征值分布分散（如高度相关）时，收敛极慢。 | **极快**。通常比LMS快一个数量级，因为自相关矩阵的逆起到了白化（正交化）作用。 |
| **计算复杂度**       | **低**。O(M)*O*(*M*)，M为滤波器阶数。每次只需乘加运算。      | **高**。O(M2)*O*(*M*2)。需要更新矩阵的逆，计算量大。         |
| **稳态误差**         | 在平稳环境下，稳态误差接近最优；受步长 μ*μ* 影响，μ*μ* 越大失调越大。 | 在平稳环境下，稳态误差通常更小，更接近最优维纳解。           |
| **数值稳定性**       | **高**。算法鲁棒，不易发散（只要步长在合理范围内）。         | **较差**。递归求逆过程容易受有限字长效应影响，可能导致自相关矩阵失去正定性而发散。 |
| **对特征值的敏感性** | **极其敏感**。最大特征值与最小特征值之比决定了收敛的困难程度。 | **不敏感**。因为直接计算了自相关矩阵的逆，相当于对输入信号进行了白化处理。 |

### 💡 总结与选择建议

- **随机梯度法（LMS）\**像是一个\**“近视眼”**，只看脚下一步，虽然走得不快甚至可能绕弯，但体力消耗极小（计算量小），非常皮实耐用，适合长期奔跑在资源有限的设备上。
- **最小二乘法（RLS）\**像是一个\**“全知视角”**，回顾历史所有数据，每一步都精打细算找到当前最优解，所以跑得极快，但极度消耗脑力（计算量大），且如果脑子算力不足出现舍入误差（数值不稳定），可能会精神崩溃（发散）。

### 1.4 自适应滤波器的典型应用

1. 未知系统建模
2. 得到未知系统的逆变换
3. 预测未来的信号
4. 删除干扰项

在标准框图中：**Plant** 指的是**需要被建模或补偿的未知系统**（也称为"被控对象"或"主路径"）。

- **物理意义**：它可以是任何具有未知特性的线性系统，比如声学通道、通信信道、机械振动系统等
- **数学表示**：通常用传递函数 *P*(*z*)  或冲激响应 *p*(*n*)  表示
- **作用**：自适应滤波器的目标就是**估计或逼近这个 Plant 的特性**

**Delay** 在自适应滤波器中有两种常见含义：

### 1. **物理路径时延（Propagation Delay）**

- 信号通过 Plant 传播所需的**物理时间**
- 例如：声音从扬声器传到麦克风需要几毫秒
- 在主动噪声控制中，这个时延会影响系统的**因果性**和**稳定性**

### 2. **参考信号时延（Reference Delay）**

在 **FxLMS（Filtered-x LMS）** 等算法中常见的结构：

- 为了让自适应滤波器能够**因果地实现**，需要在参考信号路径中加入一个**预估时延**
- 这个 Delay 补偿了 Plant 带来的相位滞后，确保滤波器输出能与噪声信号同步到达误差点

### 数学表示：

$$d(n)=x(n−Δ)  $$

其中 Δ  是时延的采样点数。

## 2. 平稳过程描述

### 2.1 平稳过程的性质

**随机过程**用于描述随时间演化的统计现象，其变化遵循概率法则。

> 关键理解：随机过程不仅仅是时间的函数，而是**一族随机变量**（每个时间点对应一个随机变量）。

**特例**：离散时间随机过程的一个具体实现就是**时间序列**： $u(n),u(n−1),…,u(n−M) $

| 矩                                 | 定义/含义                                                    | 公式                       |
| ---------------------------------- | ------------------------------------------------------------ | -------------------------- |
| **均值函数** (Mean Value Function) | 描述过程在任意时刻的平均水平                                 | $\mu(n)=E[u(n)]$           |
| **自相关函数** (Autocorrelation)   | 描述同一过程在不同时刻取值之间的线性相关性，含直流信号的原始关联 | $r(n,n-k)=E[u(n)u^*(n-k)]$ |
| **自协方差函数** (Autocovariance)  | 描述过程在不同时刻取值的协变程度（去除均值影响），去除直流信号的纯粹波动关联 | $c(n,n-k)=r-u(n)u^*(n-k)$  |

### 2.2 **严格平稳离散时间随机过程（Strictly Stationary Discrete-Time Stochastic Process）**

**严格平稳（Strict Stationarity）**：随机过程的**所有有限维分布**不随时间平移而改变。这意味着过程的统计特性在时间上是**恒定不变**的。

均值：$$\mu(n) = \mu$$

- 对于严格平稳过程，**均值必须是常数**，与时间索引 $n$ 无关
- 无论你在哪个时刻 $n$ 观察过程，其期望值都相同
- 这是时间平移不变性的直接结果：$E[u(n)] = E[u(n+k)]$ 对任意 $k$ 成立

自相关和自协方差： $$r(n, n-k) = r(k)$$，$$c(n, n-k) = c(k)$$

**解析**：

| 特性               | 含义                                                    |
| ------------------ | ------------------------------------------------------- |
| **二阶平稳性**     | 严格平稳 $\Rightarrow$ 宽平稳（若二阶矩存在）           |
| **仅依赖时差** $k$ | 不依赖绝对时间 $n$，只依赖两个时刻的**间隔**            |
| **时间平移不变**   | $r(n_1, n_2) = r(n_1-n_2) = r(k)$，其中 $k = n_1 - n_2$ |

> 注意：这里 $r(k)$ 表示滞后（lag）为 $k$ 时的自相关值。有些教材定义 $r(k) = E[u(n)u^*(n-k)]$，此时 $k$ 为正表示向后看 $k$ 个样本。

零滞后自相关 $r(0)$ 的物理意义 $$r(0) = E\left[|u(n)|^2\right]$$

- **均方值（Mean Square Value）**：信号在任意时刻的平均能量/功率
- 对于**零均值**过程：$r(0) = \sigma^2$（方差），即信号的总功率
- 对于**复数信号**：$|u(n)|^2 = u(n)u^*(n)$，包含实部和虚部的总能量
- 在信号处理中，这通常代表信号的**总功率**（直流功率 + 交流功率）

零滞后自协方差 $c(0)$ 的物理意义 $$c(0) = \sigma_n^2$$

- **方差（Variance）**：信号围绕其均值的**波动程度**
- 自协方差与自相关的关系：
  
  $$c(k) = r(k) - |\mu|^2$$

- 当 $k=0$ 时：
  
  $$c(0) = r(0) - |\mu|^2 = E\left[|u(n)|^2\right] - |E[u(n)]|^2$$

- **物理意义**：
  - $r(0)$ = 总功率（直流 + 交流）
  - $|\mu|^2$ = 直流功率（均值分量的功率）
  - $c(0) = \sigma_n^2$ = **交流功率**（纯波动分量的功率）

宽平稳（WSS）vs 严平稳

|            | 严格平稳                                | 宽平稳（WSS）                               |
| ---------- | --------------------------------------- | ------------------------------------------- |
| **要求**   | **所有有限维分布平移不变**              | 仅一、二阶矩平移不变                        |
| **均值**   | 常数                                    | 常数                                        |
| **自相关** | 仅依赖时差                              | 仅依赖时差                                  |
| **关系**   | 严格平稳 + 有限二阶矩 $\Rightarrow$ WSS | WSS $\nRightarrow$ 严格平稳（除非高斯过程） |

> **高斯过程特例**：对于高斯随机过程，WSS $\Leftrightarrow$ 严格平稳。因为高斯过程完全由均值和协方差刻画。

1. **功率谱密度（PSD）**：WSS过程的自相关函数与功率谱构成傅里叶变换对**（维纳-辛钦定理）**

   $$S(\omega) = \sum_{k=-\infty}^{\infty} r(k) e^{-j\omega k}$$

2. **线性系统分析**：WSS过程通过LTI系统后仍为WSS，输出自相关可由输入自相关与系统冲激响应计算

3. **最优滤波**：维纳滤波、卡尔曼滤波等均以WSS假设为基础

4. **谱估计**：周期图、AR模型等都需要过程的平稳性假设



### 2.2 平均各态历经

**Mean Ergodic（均值遍历性/均值各态历经性）** 是随机过程理论中一个极其重要的概念，它架起了**统计平均**与**时间平均**之间的桥梁。

核心问题：对于随机过程 $u(n)$，我们有两种定义"均值"的方式：

| 平均类型                 | 定义                                             | 含义                               |
| ------------------------ | ------------------------------------------------ | ---------------------------------- |
| **统计平均（集合平均）** | $\mu = E[u(n)]$                                  | 同一时刻 $n$，对所有样本函数取平均 |
| **时间平均**             | $\hat{\mu}_N = \frac{1}{N}\sum_{n=0}^{N-1} u(n)$ | 对**一个**样本函数，沿时间取平均   |

**关键问题**：这两种平均是否相等？

**Mean Ergodic 的定义**

若随机过程满足：

$$\lim_{N\to\infty} \hat{\mu}_N = \lim_{N\to\infty} \frac{1}{N}\sum_{n=0}^{N-1} u(n) = E[u(n)] = \mu$$

**以均方意义收敛**，即：

$$\lim_{N\to\infty} E\left[ \left| \frac{1}{N}\sum_{n=0}^{N-1} u(n) - \mu \right|^2 \right] = 0$$

则称该过程具有 **Mean Ergodic（均值遍历性）**。

> **遍历性是信号处理、通信、经济学等领域的理论基础**——否则我们无法从单次观测中推断统计特性。

Mean Ergodic 的充分必要条件

对于**宽平稳（WSS）**过程，均值遍历的充要条件：

$$\lim_{N\to\infty} \frac{1}{N} \sum_{k=-(N-1)}^{N-1} \left(1 - \frac{|k|}{N}\right) c(k) = 0$$

或等价地：

$$\lim_{N\to\infty} \frac{1}{N} \sum_{k=-(N-1)}^{N-1} c(k) = 0$$

其中 $c(k) = r(k) - |\mu|^2$ 是自协方差函数。

**直观理解**：**当时间差 $k$ 增大时，若样本之间的相关性 $c(k)$ 衰减得足够快（趋于0），则时间平均就能收敛到统计平均。**

充分条件（更易验证）

| 条件                                              | 说明                         |
| ------------------------------------------------- | ---------------------------- |
| **$\sum_{k=-\infty}^{\infty} \|c(k)\| < \infty$** | 自协方差绝对可和             |
| **$\lim_{k\to\infty} c(k) = 0$**                  | 自协方差趋于零（渐近不相关） |

> 大多数实际信号（如白噪声、ARMA过程）满足这些条件。

与 Strictly Stationary 的关系

```
严格平稳 ──────┬──────> 宽平稳（若二阶矩存在）
              │
              ├───> 均值遍历？（不一定！）
              │
              └───> 需要额外条件：自协方差衰减
              
              平稳性 ≠ 遍历性
```

**重要区分**：
- **平稳性**：统计特性不随时间变化
- **遍历性**：时间平均 = 统计平均

> 平稳过程**不一定**遍历。例如：$u(n) = A$（常数随机变量，$A$ 随样本变化）是平稳的，但不遍历——时间平均等于该样本的 $A$，不等于 $E[A]$。

更高阶的遍历性

| 类型                     | 定义                    | 应用           |
| ------------------------ | ----------------------- | -------------- |
| **Mean Ergodic**         | 时间均值 = 统计均值     | 估计直流分量   |
| **Correlation Ergodic**  | 时间自相关 = 统计自相关 | 估计功率谱     |
| **Distribution Ergodic** | 时间分布 = 统计分布     | 完整的概率推断 |

**核心公式回顾**：

$$\underbrace{\frac{1}{N}\sum_{n=0}^{N-1} u(n)}_{\text{我们能计算的}} \xrightarrow{N\to\infty} \underbrace{E[u(n)]}_{\text{我们想要的}} \quad \text{(if mean ergodic)}$$

遍历性是连接**理论模型**与**实际观测**的关键桥梁，没有它，统计信号处理将失去实践基础。

---

#### 相关矩阵

相关矩阵 **R**  是WSS过程二阶统计特性的**充分统计量**，它将一维时序的关联结构编码为有限维代数对象，使最优滤波、谱估计、模型辨识等操作成为可能。

我来深入介绍**相关矩阵（Correlation Matrix）**及其在信号处理中的核心作用。

---

## 一、基本定义

对于离散时间WSS过程 $u(n)$，取 $M$ 个连续样本构成随机向量：

$$\mathbf{u}(n) = \begin{bmatrix} u(n) \\ u(n-1) \\ \vdots \\ u(n-M+1) \end{bmatrix} \in \mathbb{C}^{M \times 1}$$

**自相关矩阵（Autocorrelation Matrix）**定义为：

$$\mathbf{R} = E\left[\mathbf{u}(n)\mathbf{u}^H(n)\right] = \begin{bmatrix} r(0) & r(1) & r(2) & \cdots & r(M-1) \\ r^*(1) & r(0) & r(1) & \cdots & r(M-2) \\ r^*(2) & r^*(1) & r(0) & \cdots & r(M-3) \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ r^*(M-1) & r^*(M-2) & r^*(M-3) & \cdots & r(0) \end{bmatrix}$$

其中 $\mathbf{u}^H(n)$ 表示共轭转置（Hermitian transpose）。

### 1. **Toeplitz结构**

$$\mathbf{R}_{ij} = r(j-i) = r^*(i-j)$$

| 特性                    | 含义                                       |
| ----------------------- | ------------------------------------------ |
| **常数对角线**          | 每条副对角线上元素相同                     |
| **由第一行/列完全确定** | 只需 $M$ 个自相关值 $r(0), \ldots, r(M-1)$ |
| **计算效率**            | Levinson-Durbin等快速算法的基础            |

### 2. **Hermitian性质**

$$\mathbf{R} = \mathbf{R}^H$$

即 $\mathbf{R}_{ij} = \mathbf{R}_{ji}^*$，矩阵等于自身的共轭转置。

### 3. **半正定性**

$$\mathbf{R} \succeq 0 \quad \text{（对所有非零向量 } \mathbf{x}, \mathbf{x}^H \mathbf{R} \mathbf{x} \geq 0\text{）}$$

**物理意义**：任意线性组合的功率必须非负。

---

### 1. **特征值边界**

$$\lambda_{min} \leq S(\omega) \leq \lambda_{max}$$

功率谱的极值被相关矩阵的特征值所界定。

### 2. **条件数与谱动态范围**

$$\kappa(\mathbf{R}) = \frac{\lambda_{max}}{\lambda_{min}}$$

| 条件数             | 含义                               |
| ------------------ | ---------------------------------- |
| $\kappa \approx 1$ | 白噪声，各向同性，自适应滤波收敛快 |
| $\kappa \gg 1$     | 有色噪声，病态矩阵，梯度方法收敛慢 |

---

## 随机模型

一个模型用来描述我们的假设。一个随机过程可以描述为一个纯随机过程的冲激加一个滤波器。

**纯随机过程：** 均值为0，方差为常数的高斯白噪声。

 随机模型中的 **AR**、**MA** 和 **ARMA** 是时间序列分析的核心工具，用于描述序列的自相关结构。它们都属于**线性随机差分方程**模型。

### 一、MA 模型（Moving Average，滑动平均）

定义：当前值是**过去白噪声冲击的加权平均**：

$$x_t = \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \cdots + \theta_q \varepsilon_{t-q}$$

其中 $\varepsilon_t \sim WN(0, \sigma^2)$ 是白噪声。简记：MA(q)

- **q**：滑动平均阶数（记忆长度）
- **核心思想**：序列的波动由外部随机冲击的累积效应造成

是平稳的（有限个白噪声的线性组合）

就像湖面波纹：丢一颗石子（$\varepsilon_t$）产生涟漪，当前水面高度是最近几次丢石子产生的波纹叠加。

### 二、AR 模型（AutoRegressive，自回归）

定义：当前值是**自身过去输出值的线性回归**：

$$x_t = \phi_1 x_{t-1} + \phi_2 x_{t-2} + \cdots + \phi_p x_{t-p} + \varepsilon_t$$

### 简记：AR(p)
- **p**：自回归阶数（依赖过去的步数）
- **核心思想**：序列具有"惯性"，当前值由历史值决定

平稳性条件：**特征方程**的根必须在单位圆外：
$$1 - \phi_1 z - \phi_2 z^2 - \cdots - \phi_p z^p = 0$$

例如 AR(1)：$| \phi_1 | < 1$ 时才平稳。

直观理解：像钟摆或气温：今天的温度与昨天、前天相关，具有"记忆性"，但冲击（$\varepsilon_t$）只影响当前时刻。

### 三、ARMA 模型（AutoRegressive Moving Average）

### 定义
**AR 和 MA 的结合**，兼具两者的特性：

$$x_t = \underbrace{\phi_1 x_{t-1} + \cdots + \phi_p x_{t-p}}_{\text{AR部分}} + \underbrace{\varepsilon_t + \theta_1 \varepsilon_{t-1} + \cdots + \theta_q \varepsilon_{t-q}}_{\text{MA部分}}$$

### 简记：ARMA(p, q)
- **p**：AR 阶数
- **q**：MA 阶数

### 算子形式（更简洁）
$$\phi(B) x_t = \theta(B) \varepsilon_t$$

其中 $B$ 是滞后算子（$B x_t = x_{t-1}$），且：
- $\phi(B) = 1 - \phi_1 B - \cdots - \phi_p B^p$
- $\theta(B) = 1 + \theta_1 B + \cdots + \theta_q B^q$

### 关键特性
| 特性         | 说明                                           |
| ------------ | ---------------------------------------------- |
| **平稳性**   | 取决于 AR 部分（$\phi(B)=0$ 的根在单位圆外）   |
| **可逆性**   | 取决于 MA 部分（$\theta(B)=0$ 的根在单位圆外） |
| **ACF/PACF** | 均拖尾（需要结合两者判断阶数）                 |

### 四、三者对比总结

| 特征         | AR(p)                        | MA(q)                          | ARMA(p,q)                             |
| ------------ | ---------------------------- | ------------------------------ | ------------------------------------- |
| **方程形式** | $\phi(B)x_t = \varepsilon_t$ | $x_t = \theta(B)\varepsilon_t$ | $\phi(B)x_t = \theta(B)\varepsilon_t$ |
| **记忆来源** | 自身历史值                   | 历史冲击                       | 两者兼有                              |
| **平稳性**   | 有条件                       | 无条件                         | 看 AR 部分                            |
| **ACF**      | 拖尾                         | q 阶截尾                       | 拖尾                                  |
| **PACF**     | p 阶截尾                     | 拖尾                           | 拖尾                                  |
| **参数估计** | 最小二乘、Yule-Walker        | 矩估计、MLE                    | MLE、条件最小二乘                     |
| **适用场景** | 有长期记忆的数据             | 短期冲击影响                   | 复杂相关结构                          |
